Para editar o construir (Build), hay que descargar Godot Engine: https://godotengine.org/download/windows/


Lógica de juego:

Al entrar un personaje, este le pasa el documento de la demanda. <br/>
El jugador la lée y marca la licencia que tiene el producto por el cual se demanda. <br/>
Luego marca las infracciones (si tiene alguna) <br/> y aprueba o desaprueba la demanda. <br/>


El juego está controlado básicamente por un script global que hace el gerenciamiento del estado.
